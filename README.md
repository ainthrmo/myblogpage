# AINN Blog Page

Multi-page portfolio blog built with React + Vite. The nav links route to dedicated pages:
Home, Journal, Projects, Notes, About, and Journey.

## Routes

- `/` Home
- `/journal` Journal
- `/projects` Projects
- `/notes` Notes
- `/about` About
- `/journey` Journey

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
